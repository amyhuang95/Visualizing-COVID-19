Background

Coronavirus disease (COVID-19) was first identified as an epidemic in Wuhan region, China, in December 2019. It soon spreads across the country and became a global pandemic.

According to the the World Health Organization, COVID-19 can spread from person-to-person contact. More countries are announcing quarantine policy that include temporary lockdown of a region and self-isolation order against citizens. These measures aim to flatten the curve, which can delay outbreak peak, decompress peak burden on hospitals, and reduce overall cases and health impacts, according to the U.S. Centers for Disease Control and Prevention (CDC).

Governments around the world and international organizations have been collecting data to monitor the disease outbreak. Notably, the Johns Hopkins University Center for Systems Science and Engineering created a publicly available data repository to consolidate COVID-19 data from various sources including the WHO, the CDC or Ministry of Health from multiple countries, and so on.

This report visualizes the up-to-date COVID-19 situation using JHUs dataset on global confirmed, death, and recovered cases, to see the international pattern of the COVID-19 disease outbreak.

Data & Preprocessing

First, we retrieve each dataset from JHUs repository and save it. URL to the repository is used to ensure the data is up-to-date every time we run the analysis.

# read in the csv files
raw.confirmed <- read_csv(url(url_confirmed))

raw.recovered <- read_csv(url(url_recovered))
# save in local computer
write.csv(raw.confirmed, file = ".\\data\\covid-19-confirmed-global.csv", row.names = FALSE)
write.csv(raw.deaths, file = ".\\data\\covid-19-deaths-global.csv", row.names = FALSE)
write.csv(raw.recovered, file = ".\\data\\covid-19-recovered-global.csv", row.names = FALSE)

Since the original dataset is in wide format, we need to reshape it into long format for plotting and performing analysis in R. To simplify the process, we use lappy function to reshape the three datasets in a time.

# combine three dataset into a list since lapply passes through a list
raw <- list(raw.confirmed,raw.deaths,raw.recovered)
reshaped <- lapply(raw, function(x){
  cleaned <- x %>% 
    pivot_longer(cols = -c('Province/State','Country/Region','Lat','Long'), 
                 names_to = "date", 
                 values_to = "cases", 
                 values_drop_na = TRUE) %>% 
    mutate(date = as.Date(date,"%m/%d/%y")) %>%  
    group_by(`Country/Region`, date) %>% 
    summarize(cum_cases = sum(cases)) 
  cleaned
  }
)
# unlist the results of lapply back to three data frames
global_confirmed <- as.data.frame(unlist(reshaped[1],recursive = FALSE))
global_deaths <- as.data.frame(unlist(reshaped[2], recursive = FALSE))
global_recovered <- as.data.frame(unlist(reshaped[3], recursive = FALSE))

The loaded data contains 187 countries in the world from 2020-01-22 to 2020-05-06.

Analysis (as of 2020-05-06)

Worldwide trend

Below plot presents the cumulative confirmed cases worldwide overtime. We can see confirmed cases started to surge after mid-March.

# annotation
who_events <- tribble(

But how bad is the situation overall? Below compared the cumulative numbers of different types of COVID-19 cases. The graph shows that the recovered cases grow far slower than infected cased, resulting in growing number of active cases. This implies the necessity of social distancing measure, such as covering face and avoiding non essential gathering.

# calculate cumulative cases for each dataset overtime
all_cases_overtime <- lapply(list(global_confirmed, global_recovered, global_deaths), function(x){
  cumulated <- x %>% 
    group_by(date) %>% 
    summarize(cum_cases = sum(cum_cases))
  cumulated
  }
)

Confirmed cases: China compared to the rest of the world

Now we want to compare confirmed COVID-19 cases in China to the rest of the world in the first three months since China became the origin of COVID-19. The annotation on the graph helps us interpret the trend. Firstly, WHO declared COVID-19 as a global health emergency when over 8,000 cases are confirmed in China. Secondly, change in Chinas reporting method explains the sudden surge of cases in mid-February. Lastly, pandemic was declared when the rest of the world had about 45,000 cases.

# subsetting & filtering data

Trend (logarithmic scale) in the US and the rest of the world as of early May

Below we added a trend line to see how fast COVID-19 spreads in the US and the rest of the world. As the graph shows, both the US and the rest of the world are growing faster than linear prediction at exponential rate (blue dashed line).

trend_us_v_world <- us_vs_world %>%
  ggplot(aes(x = date, y = cum_cases))+
  geom_line()+
  geom_smooth(method = "lm", se = FALSE, linetype = "dashed")+

According to data, the average infection fatality rate is 4%. The following table summarize 10 countries who had the highest COVID-19 infection fatality rate. The table shows the highest death rate occurred in the Nicaragua (31%), followed by MS Zaandams 22%. Recovery rate

Lastly, we want to look at the infection recovery rate to see how countries have been doing after hit by the disease outbreak. According to data, about 43.8% of infected cases have recovered from the disease on average. Table below presents countries that have mostly recovered from the outbreak, indicating low density of active cases. It also showed in the table that the situation in China has been improved with about 94% of infection recovery rate.

confirmed_recovered <- merge(global_confirmed, global_recovered, 
                         by = c("Country.Region", "date"),

recovered <- confirmed_recovered %>%
  group_by(Country.Region) %>%

Conclusions

In sum, the situation of COVID-19 outbreak is still tough globally as of 2020-05-06. Number of infected cases is growing faster than exponential rate. The average infection death rate of COVID-19 globally is higher than other historical pandemic disease. Since the average recovery rate in most countries has remained low, social distancing measures, such as wearing facial covers and keep safe distance with people, are necessary to slow down the infection and prevent another outbreak.
